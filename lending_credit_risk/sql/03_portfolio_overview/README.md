# 03 — Portfolio Overview

## Objective
Establish the headline metrics that define the scale and risk level of the loan portfolio.

## Queries Included
- Total loans, total exposure, and average loan size
- Overall portfolio default rate

## Why It Matters
These two queries produce the five KPI metrics that anchor the entire dashboard: 1.3M loans, $19B exposure, $14K average loan, 20% default rate, and $4B in loss exposure. Every segment-level finding is benchmarked against this portfolio baseline.
