# 05 — Loss Exposure Analysis

## Objective
Quantify the actual dollar loss generated by each borrower segment — moving beyond default rates to understand where financial damage is concentrated.

## Queries Included
- Loss exposure by credit grade (ordered by default rate)
- Loss exposure by DTI risk band (ordered by default rate)
- Loss exposure by loan purpose (ordered by loss exposure)

## Why It Matters
Default rate and dollar loss tell different stories. Grade G defaults at 50% but generates only $0.1B in losses due to low loan volume. Grade C defaults at 22% but drives $1.3B in losses — the largest of any grade. These queries surface the expected loss effect: **volume and exposure size determine financial impact, not default rate alone.** This distinction is the central analytical insight of the project.
